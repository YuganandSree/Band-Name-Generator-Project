
# Password-Generator-Project
This project is a command-line password generator written in Python that creates secure passwords based on user input.

A useful tool that helps users create strong and secure passwords by combining letters, numbers, and special characters.

This project is a command-line Password Generator built using Python. The program allows the user to choose how many letters, symbols, and numbers they want in their password. The program then randomly selects characters and generates a secure password based on the user’s preferences.

The generated password is shuffled so the characters appear in a completely random order, making the password stronger and harder to guess.

The project demonstrates the use of Python fundamentals such as loops, lists, user input handling, and random number generation.

## Features

User can customize the password by selecting:

Letters → alphabet characters  
Symbols → special characters such as ! # $ % &  
Numbers → digits from 0–9  

The program then:

- Randomly selects characters from predefined lists  
- Combines them into a password  
- Shuffles the characters to ensure randomness  
- Displays the final generated password
